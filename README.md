# Attendance Management System

Web client used to monitor attendance trends, run quick stats, and manage student records.

## Packages

- React 18 + TypeScript
- Vite build tooling
- Material UI + TailwindCSS
- Zustand + TanStack Query for state and server cache
- MUI X-Charts & Rsuite Table for data visuals
